## 3. API \u4ed5\u69d8\uff08\u30c9\u30e9\u30d5\u30c8\uff09

### POST /api/v1/solve
- Body: `{ "question_id": number, "prompt": string, "model": "gemini-1.5-pro" }`
- 200 OK: `{
  "question_id": number,
  "prompt": string,
  "model_vendor": "gemini",
  "model_name": "gemini-1.5-pro",
  "ai_output": string,
  "answer_number": number | null,
  "score": number,
  "evaluation": { "mode": "exact|tolerance|fail", "tolerance": 0.01 },
  "elapsed_ms": number,
  "saved": true
}`
- 4xx/5xx: \u30a8\u30e9\u30fc\u30d5\u30a9\u30fc\u30de\u30c3\u30c8\u7d71\u4e00 `{ code, message, detail? }`

### GET /api/v1/leaderboard
- Query: `period=day|week|all`, `limit=50`
- 200 OK: `[{ user_id, username, best_score, attempts, last_at }]`

### GET /api/v1/scores
- Query: `user_id?`, `from?`, `to?`, `limit?`
- 200 OK: `[{ id, question_id, score, prompt_len, created_at }]`
